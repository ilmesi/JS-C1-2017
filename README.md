# JS Basics (JS-C1-2017)


## Class content


- Node envs
- Package managers
- Module management
- ES6


## Run scripts


```bash
$ npm run es5
$ npm run es6
```


## Run api scripts

```bash
$ npm install -g json-server
$ json-server --watch api/db.json

$ npm run es5-api
$ npm run es6-api
```
